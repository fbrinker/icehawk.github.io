ICEHAWK FRAMEWORK

Lightweight PHP microframework respecting CQRS

ICEHAWK 2 is PHP 7+

Written in PHP 7 - Full test coverage

Get started

Install the ICEHAWK routing framework via composer:

Find latest stable release on out Github release section. 

{
  ...
  "require": {
    "icehawk/icehawk": "^2.0"
  }
  ...
}

Components

ICEHAWK

The basic routing framework for websites and APIs.

SESSION

The session wrapper component with data mappers.

FORMS

The CSRF safe form data transfer component.
